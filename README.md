## Blowfish
**Blowfish** - криптографический алгоритм, реализующий блочное симметричное шифрование с переменной длиной ключа.    
*Размер ключа*: от 32 до 448 бит    

*Размер блока*: 64 бита    
*Число раундов*: 16    
*Тип:* Сеть Фейстеля
